# Documentacao do PyCQLog

Esta pasta contem a documentacao ativa do `PyCQLog`.

O `PyCQLog` e um logger para radioamadorismo com foco em operacao pratica, propriedade local dos dados e facilidade de uso, tanto em modo desktop quanto em modo daemon com interface remota.

O projeto e uma iniciativa de Claudio - PY9MT.

## O que o projeto busca entregar

- registro rapido e confiavel de QSOs
- uso simples para operador individual
- evolucao gradual para estacoes maiores, servidores pequenos e Raspberry Pi
- integracoes com ferramentas comuns do radioamador, como ADIF, JTDX, WSJT-X, Club Log, QRZ e LoTW
