# Documentacao do Sistema Python

Esta pasta contem a documentacao ativa do `PyCQLog`, a nova aplicacao desktop em Python com PyQt.

O projeto nasce da necessidade de um aplicativo de log para radioamadorismo escrito em uma linguagem mais moderna, rodando em Linux, e com uma base tecnica mais aberta a colaboracao comunitaria e evolucao conjunta.

O projeto esta aberto a todos que queiram contribuir, desenvolver, usar no dia a dia, testar, reportar erros, sugerir melhorias, revisar codigo, melhorar a documentacao ou ajudar na manutencao ao longo do tempo.
